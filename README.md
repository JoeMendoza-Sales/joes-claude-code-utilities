# Joe's Claude Code Utilities

A collection of useful utilities and enhancements for Claude Code.

## Utilities

| Utility | Description |
|---------|-------------|
| [Context Monitor](context-monitor/) | Live color-coded context usage meter in the status bar with threshold alerts. Never lose work to context overflow again. |

## Install

Each utility has its own README with setup instructions. Click through to the one you want.

## Prerequisites

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code)
- `jq` and `bc` (`brew install jq bc` on Mac)
